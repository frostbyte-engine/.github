## frostbyte

creation date: Jun 25, 2025

![](../repoassets/showcase1.png)

frostbyte is an attempt to make a game engine as close to Roblox as possible (at least from a scripting perspective).

* [frostbyte](https://github.com/frostbyte-engine/frostbyte) houses the engine code.
* [frostbyte-desktop](https://github.com/frostbyte-engine/frostbyte-desktop) houses the desktop application.
